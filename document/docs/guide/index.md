## 环境支持

DW UI 可以在支持 ES2018 和 ResizeObserver 的浏览器上运行。

由于 Vue 3 不再支持 IE11，Element Plus 也不再支持 IE 浏览器。

## 使用包管理器

```shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install dw-ui --save

# Yarn
$ yarn add dw-ui

# pnpm
$ pnpm install dw-ui
```
如果您的网络环境不好，建议使用相关镜像服务 cnpm 或 中国 NPM 镜像。
