# dw_ly_component

vue3 组件库


# 目录

```shell
build       # 打包模块
document    # 文档模块
example     # 使用示例模块
package     # 组件源码模块
dist        # 打包后的文件
```

# 使用方法

在跟目录下  使用 `pnpm i` 安装依赖

随后使用哪个子模块 就进哪个子模块 使用 `npm i` 安装依赖

```shell
npm run doc     # 运行组件文档库
npm run dev     # 运行组件使用示例库
npm run build   # 打包组件库      提前全局安装下sass  npm i -g sass
```
